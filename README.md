# Flow Browser

![flow browser screenshot](https://gcdnb.pbrd.co/images/CG4GOGlNUAZ9.png?o=1)

A minimal open source web browser made with Python.

## Features

- Tabbed browsing
- Address bar with URL navigation
- Back, Forward, and Reload buttons
- Collapsible sidebar menu with browser features
- Dark/Light theme toggle (browser UI + website dark mode detection)
- Home page navigation
- Developer tools (Inspect) - opens in new tab
- Settings dialog with theme, web dark mode, and home page configuration
- Bookmarks management (add, open, remove)
- Browsing history tracking and viewing
- Downloads management (placeholder)
- New tab creation
- Mouse pointer lock support for gaming >:)
- Sick SICK UI

## Requirements

- Python 3.12+
- PyQt6
- PyQt6-WebEngine
