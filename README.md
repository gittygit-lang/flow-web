# Flow Browser

![flow browser screenshot](https://gcdnb.pbrd.co/images/Tdney1MVctPo.png?o=1)

A minimal open source web browser made with Python.

## Requirements

- Python 3.12+
- PyQt6
- PyQt6-WebEngine

## Installation

- pip install pyqt6
- git clone https://www.github.com/gittygit-lang/flow-web.git
